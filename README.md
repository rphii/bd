# BD
Simple, one-file build tool for C projects.

## Windows
1. Put [`bd.c`](src/bd.c) into the root of your project, configure settings if needed
2. Run `gcc -o bd bd.c` or `gcc -o bd bd.c; ./bd.exe`
3. Run `./bd.exe -h` for help
